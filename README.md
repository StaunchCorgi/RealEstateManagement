# Real Estate Management API

## Features
- JWT Authentication
- Role-based authorization
- Agent access requests
- Property inquiries
- Admin approval workflow

## Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- JWT Authentication

## Setup

2. Update appsettings.json
3. Run migrations
4. Start API
